## 左移（<<）
- m << n 等于 m * pow(2, n)
  - 比如：
  - 1 << 0 = 1 * pow(2, 0) = 1 、1 << 1 = 1 * pow(2, 1) = 2、 1 << 2 = 1 * pow(2, 2) = 4
  - 2 << 0 = 2 * pow(2, 0) = 2 、2 << 1 = 2 * pow(2, 1) = 4、 2 << 2 = 2 * pow(2, 2) = 8
  - 3 << 0 = 3 * pow(2, 0) = 3 、3 << 1 = 3 * pow(2, 1) = 6、 3 << 2 = 3 * pow(2, 2) = 12


## 右移（>>）
m >> n 等于 parseInt(m / pow(2, n))，只要整数部分，舍去小数。      
是/，不是%     
%：a % b = 【a除以b的余数】。
